HDB Resale Flat Prize Analyser in Python

This project is designed to observe the trends and key factors influencing the Housing and Development Board(HDB) resale price in Singapore. With a user-friendly web interface, it empowers users with actionable insights, serving as a practical navigational aid in the housing landscape.

Done by: Shahul Hameed Muhammed Irshath, Tan Rui Yang, Ong Yu Xiang, Daniel Toh, Aziel Khong
